💡 Prompt pour générer un quiz avec une IA

Copie ce prompt dans ChatGPT, Claude ou Gemini, et remplace le sujet à la fin.

Tu es un générateur de QCM pédagogiques. Génère un quiz sur le sujet suivant : [SUJET ICI] Retourne UNIQUEMENT un JSON valide (sans texte avant ni après, sans balises markdown) avec cette structure exacte : { "title": "Titre du quiz", "subject": "Matière / domaine", "description": "Brève description du quiz", "for": "Prénom", "replay": false, "questions": [ { "id": 1, "type": "single", "text": "Question à choix unique", "options": ["Option A", "Option B", "Option C", "Option D"], "correct": [0], "explanation": "Explication de la bonne réponse" }, { "id": 2, "type": "multiple", "text": "Question à choix multiples", "options": ["Option A", "Option B", "Option C", "Option D"], "correct": [0, 2], "explanation": "Explication des bonnes réponses" }, { "id": 3, "type": "word", "text": "Question à réponse courte — un seul mot ou expression brève attendue", "options": [], "correct": ["réponse attendue", "variante acceptée"], "explanation": "Le mot exact attendu (insensible à la casse)" }, { "id": 4, "type": "text", "text": "Question ouverte — UNIQUEMENT si vraiment nécessaire. Maximum 1 par quiz.", "options": [], "correct": [], "explanation": "Éléments clés attendus dans la réponse" } ] } CHAMP "for" : indique le prénom de l'élève concerné par ce quiz. Prénoms disponibles : Zoé, Timéo, Tom, Noé Si le quiz est pour tout le monde, omets le champ "for". RÈGLES : - Types disponibles : "single", "multiple", "word", "text" - "single" : 1 seule bonne réponse, "correct" = [index] - "multiple" : plusieurs bonnes réponses, "correct" = [index, index, ...] - "word" : réponse courte auto-corrigée, "correct" = ["mot", "variante"] en minuscules - "text" : réponse longue manuelle — MAXIMUM 1 PAR QUIZ, à éviter - Génère le nombre de questions demandé (par défaut 8 si non précisé) - "replay": true si le quiz doit pouvoir être refait plusieurs fois, false sinon (défaut) - Favorise "single" et "multiple" - "word" uniquement pour réponses très courtes et précises Sujet : [REMPLACE ICI PAR TON SUJET] Pour : [PRÉNOM ou "tout le monde"]
